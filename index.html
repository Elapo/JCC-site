Bijleren

In de R Java Community zitten we geen moment stil wat betreft bijleren en doorgroeien.
Er is waarschijnlijk geen andere firma die zoveel mogelijkheden biedt voor eenieder om zijn kennis uit te breiden, een greep uit het aanbod: Brains sessies, E-library en papieren boeken, een waslijst van real-world technische trainingen via ons eigen Education Center, soft-skill trainingen, methodologie trainingen, AcADDemICT-traject, certificatie trajecten, group-trainings, Java Café’s ©, coaching on the job, lidmaatschap bij vakgroepen (zoals BeJUG), Squads, Community Tech days, Community Leaders Team, …
Het is bijna te veel om op te noemen, opgepast voor een brain-overload!

Personal Development

Zonder uitzondering heeft iedereen in onze Java Community trouwens een Personal Development Plan dat doorheen het jaar wordt uitgevoerd en opgevolgd. Dit PDP vormt het kader om te bespreken wat ieders individuele ambities zijn, wat er nog ontbreekt om er te geraken, (de fit-gap), en welke stappen er dan concreet genomen moeten worden om je ambities te realiseren. Dit leggen we dan vast in een helder plan, en je people manager ondersteunt je bij iedere stap. Deze stappen kunnen tal van dingen inhouden (zie hierboven), maar 1 zaak staat al vast: je zal je doorheen je carrière bij Realdolmen geen moment vervelen!

Tools

Weet ook dat we de beste tools hebben om ons leven makkelijker, innovatiever, socialer, en simpelweg ook leuker te maken. Zo hebben we bijvoorbeeld Yambla: onze super-leuke tool om ideeën om te zetten in praktijk. Of Yammer: ons Enterprise Social Network, Lync: om te chatten, bellen, screensharen en online vergaderen met al je collega’s zonder limieten. Of wat dacht je van ons eigen Java community platform o.b.v. de alom geprezen Atlassian stack? Hierin zitten best-of-breed Java tools zoals Jira, Confluence, Bamboo, Fisheye, Crucible, etc.
